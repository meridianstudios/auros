# Auros Live — retro (90s) font

The "90s Classic" style looks for a font named **Star4000** here. It's optional:
without it the style falls back to a clean system monospace, so nothing breaks.

To get the authentic blocky WeatherStar 4000 look, drop the free fan-made font
into this folder as either (or both):

- `Star4000.woff2`  (preferred — smallest)
- `Star4000.ttf`

The font is widely distributed for exactly this kind of tribute (search
"Star4000 font" — the TWC-fan community maintains free recreations). Convert a
`.ttf` to `.woff2` with any online converter if you only have the TTF.

Once the file is here, the retro style picks it up automatically on next load —
no code change needed. The `@font-face` is already declared in AurosLive.css.
